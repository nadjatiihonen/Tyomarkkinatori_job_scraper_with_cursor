# jobmarket-scraper

Simple script that collects job listings from Tyomarkkinatori and saves them to Excel.
Yksinkertainen skripti, joka kerää työpaikat Tyomarkkinatorilta ja tallentaa ne Exceliin.

Run / Aja:
- `python jobmarket-scraper.py`

## Technologies / Teknologiat

- Python 3.9+
- Playwright (browser automation / selainautomaatio)
- pandas (table processing / taulukon käsittely)
- openpyxl (Excel read/write / Excel luku-kirjoitus)
